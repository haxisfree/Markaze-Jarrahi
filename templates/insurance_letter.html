I'm building web-based, data-driven apps using Django.
Eventually (or unfortunately), I will need to generate some reports that are printer-friendly. Logically, PDF is the format for such files...
so how am I going to convert my xHTML and CSS to a nice-looking PDF document?
The Django Book has a whole chapter dedicated to Generating Non-HTML Content.
They seem to to be fond of ReportLab ToolKit.
The caveat here, though, is that you need to know a bit about the internals of a PDF document.
If you're familiar with this, the ReportLab toolkit seems to be the way to go! It has many features, and it seems to be a powerful PDF-generating tool.
